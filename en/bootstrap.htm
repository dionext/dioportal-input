Overview

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components. https://en.wikipedia.org/wiki/Bootstrap_(front-end_framework)

Official site: https://getbootstrap.com/
